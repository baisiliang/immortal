# 技艺与生活技能设计

## 一、总则

- **两大类**：制造、研学。
- **兴业方向 / 职业**：每类下有若干方向，**每个方向自成一个职业，共 12 个**（制造 6 + 研学 6）。
  - 职业名**后定**（暂用方向名，如锻造、制药、琴、棋……）。
  - 其中「**制茶 + 酿酒**」合为同一个方向 / 职业。
- 每个方向包含：**技能（主动 / 地图 / 非战时复用）+ 被动 + 经营建筑**。
- **经营建筑**：可在家园（村庄）建造，**每月结算产出一次**；各建筑**具体特色后定**。
- **技艺资质**决定学习速度与上限；方向可随等级**解锁 / 强化**（进阶可选学）。

> ★ = 来自设计稿；○ = 待补 / 建议，可替换。

## 二、制造类 · 兴业方向

> 被动统一方向：都与**制品的效果 / 词条 / 等级**有关。建筑多为**产出型**（每月结算量产）。

| 方向 | 技能 / 产物 | 经营建筑 |
| --- | --- | --- |
| ★锻造 | 打造兵器、防具（含锻造手法特殊词条，见道具模块） | 待定特色 |
| ★制饰 | 制造饰品（腰带、香囊、戒指） | 待定特色 |
| ★制药 | 疗伤药、解阴煞侵蚀药等 | 待定特色 |
| ★制毒 | 毒药、淬毒 | 待定特色 |
| ★做菜 | 餐食（非战时复用，提供增益 / 减益） | 待定特色 |
| ★制茶酿酒 | 茶（安神 / 恢复神思 / 清心）+ 酒（短期加成，结束后有负面，可作名贵礼品） | 待定特色 |

## 三、研学类 · 兴业方向

> 修心 / 社交 / 研学 / 玄修。建筑侧重**个人被动增益 / 训练加速**；道法·炼丹为产出型。

| 方向 | 技能 | 被动 | 经营建筑 |
| --- | --- | --- | --- |
| ★琴 | 抚琴（地图技能：按琴技提升所在位置人物好感，**仅低好感时有效**） | 提高神（神思）恢复 / 神上限 | 待定特色 |
| ★棋 | 对弈（邀请互动，小游戏，可下注） | 小游戏特殊卡牌 / 特殊技能 | 待定特色 |
| ★书 | 修补书籍残页（**可修补功法残篇**，呼应门派传承） | 研学效率加成 / 阅历获取加成 | 待定特色 |
| ★画 | 作画（给人作画可送礼提升好感） | 提高魅力 / 送礼好感加成 | 待定特色 |
| ★炼丹（道法） | 炼丹（恢复健康、提高寿元等特殊效果） | 延寿 | 待定特色（产出型） |
| ○佛法 | 诵经 / 禅修（净化阴煞、降心魔、安神） | 抗心魔 / 心境稳固 / 降疯魔值 | 待定特色 |

## 四、经营建筑机制

- **每月结算产出一次**；各建筑具体特色**后定**。
- 制造类建筑多为**产出型**（量产制品）；研学类建筑多为**个人被动增益 / 训练加速**；炼丹（丹房）为产出型。
- 建筑可随等级 / 资源升级；家园（村庄）= **各方向建筑的集合**（详见村庄经营模块）。

---

## 五、与其他系统衔接

- **道具与装备**：制造类产出兵器 / 防具 / 饰品 / 药 / 毒 / 餐食 / 茶 / 酒。
- **村庄经营**：各方向建筑构成家园产业，可远程运营。
- **功法 / 门派**：书·修补残页 ↔ **功法残篇**（门派传承习得的功法可能带残篇，书可修补；地区恩义换取的功法无残篇）。
- **好感 / 社交**：琴、棋、画服务好感与互动；事件联动。
- **世界 / 净化**：制药、炼丹、佛法与"解阴煞 / 降心魔 / 延寿"主题挂钩。
- **基础行为**：采集（速取材料）、读书（通用养成，提悟性 / 阅历、解锁高阶功法）仍保留。
