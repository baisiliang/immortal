# 经济与交易设计

## 一、基底

- 以**宋朝经济**为基础（商业繁荣、市集发达、有纸币交子），在上层做**架空改造**。

---

## 二、货币体系

- **多级货币**：铜钱 / 银两 / 金，分级计价（兑换比率待定）。
- 可选**纸币（交子）**作为大额支付（待定）。

---

## 三、定价机制

- 商品**基础定价基本固定**（不做大幅供需波动，保持轻松）。
- **折扣 / 溢价**来自：
  - 与**商人的好感**（见 `09_人物关系`）。
  - 人物**魅力**等属性（见 `人物属性设计.md` 状态属性）。
  - 特定**事件**（行情 / 机缘等）。

---

## 四、交易场所

- 各区域 / 门派的**店铺 / 集市**买卖。
- **行商 / 特殊商人**：贩售稀物 / 限定货。
- **黑市**（待定）：销赃 / 黑货（与盗窃等负面手段联动）。

---

## 五、产销平衡

- **产出端**：村庄经营产出、制造类制品（见 `07_村庄经营`、`06_技艺系统`）。
- **消耗端**：购买材料 / 丹药 / 装备、村庄建设等。
- 钱货循环作为养成资源的一环；**功法主要靠恩义 / 交友 / 传承**，不直接用钱买（见 `08_世界地图`）。

---

## 六、与其他系统衔接

- **人物关系**：商人好感、魅力 → 折扣 / 溢价。
- **村庄 / 技艺**：产出销售换钱，材料采购投入制造。
- **道具与装备**：买卖兵器 / 防具 / 饰品 / 药材 / 餐食 / 酒。

---

## 待定

- 货币兑换比率、是否启用交子。
- 店铺 / 集市 / 行商 / 黑市清单与分布。
- 折扣 / 溢价公式；各类物品基础定价。
- 是否引入地区物价差（低买高卖）。
