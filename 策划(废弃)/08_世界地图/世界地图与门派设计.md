# 世界地图与门派设计

## 一、地图结构

- **地理参考**：以**重庆版图**为参考；选取带**神话传说**的地点关联故事、建立门派。
- **题材**：以**古代传说**为主，不用现代内容；可有少量**外来门派**（如苗族巫蛊），占比不宜多。
- **侵蚀度同心圆**：越靠近**酆都**侵蚀度越高；区域按侵蚀度产生**对应等级的事件 / 敌人**。
- **移动**：区域内移动不耗时间；**跨区域移动耗时（耗神思）**（见 `04_时间与回合`）。
- **驿站**：花费资源建立**驿站**后即可前往对应区域；**酆都需主线开启**——击败酆都附近的**六大鬼王**才能开启酆都区域。
- **净化**：解决区域事件可**降低侵蚀度**（有下限），但一段时间后会**逐渐回升**到原本等级（见 `00_核心`）。

## 二、门派

- **共 18 个**；各有**立场**（名门正派 / 邪派 / 亦正亦邪）与**地域神话特色**。
- **一世只能加入一个门派**。

## 三、关系 / 好感（双层）

- **地区 / 门派恩义**：对门派的**贡献值**，主要靠完成**门派区域事件**积累；用于**兑换功法 / 资源**。
- **个人好感**：对**具体 NPC** 的好感，影响**交友 / 送礼 / 事件**；可通过作画 / 礼品 / 琴棋互动等提升（见 `06_技艺系统`）。
- **一世一门**：加入门派后排他。

---

## 四、功法获取

- 门派**敝帚自珍**，不轻易外传功法。
- 途径：
  1. **地区恩义兑换**（**无残篇**）。
  2. **交友获得**。
  3. **盗窃**（负面手段）。
  4. **加入门派传承**：按门派等级有**传道师**，**消耗门派贡献**获取传承（**可重复**）；学到的功法依**传道师等级 + 资质**，**可能带残篇章**。
- 残篇可由**书职业·修补残页**修补（见 `06_技艺系统`）。

---

## 五、与其他系统衔接

- **时间**：跨区域移动耗时；驿站缩短可达性。
- **核心循环**：侵蚀度 / 六鬼王 / 酆都为主线骨架（见 `00_核心`）。
- **技艺**：送礼（画 / 礼品）、书修补残篇、对弈交友。
- **村庄**：驿站与远程运营联动。
- **事件 / 剧情**：区域事件、门派事件、奇遇。
