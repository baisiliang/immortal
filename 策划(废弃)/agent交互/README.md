# Agent 交互归档

本目录记录与 AI 协作过程中的**关键决策与结论**，按主题分子文件夹归类。目的：

- 后续交互**只需读取相关主题的记录**，无需把全部历史传入上下文。
- 决策留档、可追溯、方便查阅。

## 项目约定

- **临时用词**：现有"出招速度""招式"等为临时表述；**古风化 / 底蕴化的统一命名放到设计最后**统一处理。
- **初期实现**：先用系统 / 简易**占位资源**跑通流程，后期再做美术调优。
- **文档约束**：不出现其他游戏名 / 其他游戏内容。
- **精气神最终映射**：精（精血 / HP）、气（真气 / 蓝）、神（神思 / 行动力）；阴煞侵蚀独立于精气神，由「心魔 / 疯魔值」承载。
