# 属性与资质 · 决策记录

## 已定

- **底层六属性**：风、火、雷、水、木、土；阳（天）= 风火雷，阴（地）= 水木土；混元 = 均衡；无属性 = 锻体 / 体修。
  - **属于底层设计概念，不直接展示给玩家**；面向玩家用武侠味别名（如「玄阴」「金刚」）。
  - 基调：**武侠为主、仙侠为辅**。
- **精气神（三元核心资源，最终映射）**：
  - **精（精血）= 血条（HP）**。
  - **气（真气）= 蓝条（MP）**：战斗能量，施展招式 / 功法、护体消耗，可调息恢复。
  - **神（神思）= 行动力 / 精力**：战斗外日常行动消耗（读书 / 锻造 / 赶路等）；**与阴煞解耦**。
  - **阴煞侵蚀 / 疯魔**由独立的「**心魔 / 疯魔值**」承载（状态属性），不绑精气神。
  - 护盾：功法 / 装备提供，优先于精血承伤；神魂 / 直接伤害无视护盾防御。
  - 命名沿革：曾定"精=行动力/气=HP/神=蓝"，后翻转为更贴武侠的本映射。
- **主属性（暂定 4 个，先天底层、决定成才，不含后天战斗属性）**：
  - 悟性（研习效率）、根骨（体魄气血根基）、定力（心神 / 神上限 / 抗心魔）、福源（机缘气运）。
- **次要属性**：阅历（影响研学能力）。
- **状态属性**：
  - 创建时决定：性格（心性）、容貌 / 魅力。
  - 动态：心情、名誉、心魔 / 疯魔值、伤势（**整体伤、不分部位**）、寿命。
- **资质**：武功资质 + 技艺资质（门类待定）；属性相性影响修炼速度与威力。

## 讨论备注

- 魅力、心性曾被考虑为主属性，最终归入**状态属性**。
- 经脉 / 气感 / 体魄因与已有概念重复被排除；主属性暂定 4 个。
