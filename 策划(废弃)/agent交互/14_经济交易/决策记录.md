# 经济与交易 · 决策记录

## 已定

- **基底**：宋朝经济（商业繁荣、市集、交子），上层架空改造。
- **货币**：多级（铜钱 / 银两 / 金），可选纸币（交子），比率待定。
- **定价**：**基础固定**，无供需波动；**折扣 / 溢价**来自商人好感、人物魅力、特定事件。
- **交易场所**：店铺 / 集市、行商 / 特殊商人、（待定）黑市销赃。
- **产销**：村庄 / 制造产出 → 售卖；材料 → 制造；**功法不直接用钱买**（靠恩义 / 交友 / 传承）。
