# 事件与剧情 · 决策记录

## 已定

- **叙事基调**：渐进式揭真相、开场守护家园、反"越强越难"（玩家变强=世界向好）。
- **多层并行事件**：主线 + 区域事件（按侵蚀度分级）+ 门派事件（恩义 / 贡献）+ 随机奇遇（高价值 + 抉择）+ NPC / 社交事件（好感）。
- **触发**：按地点 / 月结算 / 好感 / 侵蚀度 / 进度。
- **主线脉络**（轻松化重做）：开场守护 → 渐入真相（器灵渎职放祟）→ 净化历练（唤醒解救·暖反馈）→ 六关守敌（邪祟爪牙）→ 酆都终局（重新封印 / 收服邪祟）→ 神器碎片。
- **抉择与后果**：正 / 负手段（含盗窃），影响好感 / 恩义 / 名誉 / 心魔；惩罚不过重。

## 剧情轻松化重做（推翻旧地府暴乱设定）

- **起因换新**：地府神器封印邪祟 → 器灵（主角前身）**渴望自由、擅离职守** → 邪祟逃脱、神器碎裂 → 阴煞悄然侵世（**影响暂有限**，故仙人 / 地府不直接出手，由转世器灵补救）。
- **删除旧沉重设定**：地府大战 / 暴乱、生死簿毁、孟婆桥断、轮回停摆、群鬼出逃、嗜血嗜杀。
- **侵蚀表现**：痴狂失魂、**不嗜血嗜杀**、**可净化唤醒解救**（暖反馈）。
- **基调**：成长 + 担当 + 修补；主角前世过错出于渴望自由，旅途中释怀。
- 世界观内核未变：阴煞侵蚀、器灵转世越轮回越强、18 门派、酆都为终局。
