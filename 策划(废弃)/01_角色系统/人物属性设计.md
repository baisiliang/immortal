# 人物属性设计

> 资源类数值（精 / 气 / 神、护盾）见 `属性.md`；资质天赋见 `资质与天赋设计.md`。

## 一、主属性（四维，先天底层，决定成才）

> 定位：**决定人物成才上限的底层数据线**，先天为主、出生大致定型（仅少数丹药 / 奇遇 / 神器碎片可微调）。不含"身法 / 力道"等后天战斗属性——那些由功法、装备、境界派生。

| 主属性 | 定位 |
| --- | --- |
| **悟性** | 领悟与学习效率（研习功法 / 技艺学得快慢） |
| **根骨** | 体魄；**精血（HP）与真气根基**、外功根基 |
| **定力** | 心神稳固：**神思（精力）上限**、抗心魔 / 抗阴煞、专注 |
| **福源** | 机缘气运：奇遇、掉落、事件好运 |

## 三、状态属性

> 部分在**创建人物时决定**（相对固定），部分**动态变化**。先列主要项，后期完善。

**创建时决定（相对固定）**
- **性格（心性）**：影响事件抉择、正邪倾向、某些功法 / 事件门槛。
- **容貌 / 魅力**：影响社交、人缘、门派好感、交友与招揽。

**动态变化**
- **心情**：影响行动效率与事件触发。
- **名誉**：声望，影响门派与 NPC 的态度、可触发的事件与交易。
- **心魔 / 疯魔值**：**承载阴煞侵蚀的唯一指标**（精气神与之解耦）；接触阴煞 / 深入侵蚀区会累积，过高会异化（呼应世界观"被侵蚀者状若疯魔"）。
- **伤势**：**整体伤，不区分身体部位**。
- **寿命（年龄 / 寿元）**：统一寿命系统（见 `04_时间与回合`）。

> 备注：状态属性后期再补充细化（数值范围、相互影响、恢复方式等）。
