# 事件与剧情设计

## 一、叙事基调（已定，见 `世界观.md`）

- **渐进式叙事**：不一上来揭示苦大仇深的真相，通过事件**一点点揭开主角身世与世界真相**。
- **开场**：村庄被养育、长者引导、觉醒、踏上守护家园 / 救世之路。
- **反"越强越难"**：玩家变强 = 世界向好（主线推进，侵蚀度下降）。

---

## 二、事件分层（多层并行）

| 层级 | 触发 | 作用 |
| --- | --- | --- |
| **主线事件** | 进度 / 关键节点 | 揭示真相、推进净化、六鬼王 / 酆都、神器碎片 |
| **区域事件** | 地点 + **侵蚀度**分级刷新 | 完成可**降低侵蚀度**、产出奖励 |
| **门派事件** | 门派区域 / 加入状态 | 积累**恩义 / 门派贡献**、换功法 |
| **随机奇遇** | 探索 / 时间 / 概率 | 高价值奖励与**抉择** |
| **NPC / 社交事件** | **好感** / 时间 / 地点 | 交友、送礼、人物支线 |

> 按**地点 / 月结算 / 好感 / 侵蚀度 / 进度**等条件触发。

---

## 三、主线脉络（框架）

1. **开场**：家园、村长、觉醒净化之力、立志守护。
2. **渐入真相**：通过事件逐步揭开**器灵身世（前世渴望自由、渎职放祟）**、阴煞起源、酆都邪祟。
3. **净化历练**：各区域净化被侵蚀者（**唤醒解救**、暖反馈），积累实力与神器碎片。
4. **六关守敌**：邪祟爪牙 / 被阴煞侵蚀的六大凶物镇守酆都外围（主线 gate）。
5. **酆都终局**：直面逃脱的邪祟，**重新封印 / 收服**，补救世界。
6. **神器碎片**：沿途收集，解锁 / 强化器灵之力（见轮回成长线）。

---

## 四、抉择与后果

- 事件含**抉择**，可走**正面**（交友 / 助人 / 净化）或**负面**（盗窃等）手段，影响好感 / 恩义 / 名誉 / 心魔等。
- 抉择服务"轻松但有取舍"的体验，避免惩罚过重。

---

## 五、与其他系统衔接

- **世界 / 门派**：区域事件挂侵蚀度，门派事件挂恩义 / 贡献。
- **时间**：事件按**月结算**生成 / 推进（见 `04_时间与回合`）。
- **关系 / 技艺**：社交事件挂好感，送礼（画 / 礼品）、对弈交友联动。
- **轮回**：主角死亡 → 轮回事件 → 神器碎片加成（见轮回成长线）。
