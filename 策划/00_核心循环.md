# 核心循环

一款中式风格的御兽战斗游戏，玩法由三大模块构成：**Metroidvania 探索**、**灵兽收集养成**、**策略回合制战斗**。三者不设单一主轴，互相喂养，形成**均衡环形循环**。

## 三大模块与互喂关系

| 模块 | 核心体验 | 产出（喂给其他模块） | 消耗（依赖其他模块） |
|------|---------|--------------------|--------------------|
| **探索** | 横版跳跃、地图解锁、隐藏收集、洞天福地 | 遇敌机会、灵兽蛋、谜题道具、新区域新灵兽、造化赐福地点 | 需携带特定灵兽（能力）/达成实力门槛/解谜才能推进 |
| **战斗** | 3V3 站位、属性克制、回合策略 | 评分星级掉落（蛋/材料/特殊道具）、捕捉机会、灵兽与驭兽师经验 | 队伍战力、技能树、驭兽师加点技能 |
| **收集养成** | 捕捉/孵化、品阶成长、进化、造化、队伍搭建 | 提供探索移动能力、提供战斗战力、驭兽师成长 | 驭兽空间限制、依赖战斗/探索产出的资源 |

## 循环图

```
                    ┌─────────────────────────────────────┐
                    │            探索（Metroidvania）        │
                    │  混合解锁：中式灵兽能力(火眼金睛看破虚妄/  │
                    │  特定地形)＋Boss实力门槛＋解谜道具门槛     │
                    │  洞天福地 → 灵兽造化赐福地点              │
                    └───────────┬───────────────▲───────────┘
        产出：遇敌/蛋/谜题道具/    │               │ 消耗：携带特定灵兽
        新区域新灵兽/赐福地点      ▼               │ 才能开路（队伍构成影响探索）
                    ┌─────────────────────────────────────┐
                    │          策略回合制战斗（3V3站位）       │
                    │  每战评分：回合数＋伤亡 → 星级 → 掉落概率   │
                    │  特殊方式击败特定灵兽 → 概率掉特殊道具      │
                    │  打残血＋契约 → 捕捉                     │
                    └───────────┬───────────────▲───────────┘
        产出：星级掉落(蛋/材料/    │               │ 消耗：队伍战力/技能树
        特殊道具)＋捕捉机会＋经验   ▼               │ ＋驭兽师加点技能
                    ┌─────────────────────────────────────┐
                    │            收集 · 养成                  │
                    │  来源：捕捉(占驭兽空间)＋孵化            │
                    │  驭兽空间：数量上限＋可驾驭品阶上限         │
                    │  品阶成长 / 进化(血脉) / 造化(洞天赐福)    │
                    │  驭兽师：修为境界＋加点(被动/主动技能)      │
                    └─────────────────────────────────────┘
        闭环：收集养成 → 提供探索移动能力 & 战斗战力 → 评分掉落/捕捉 → 回到收集养成
```

## 设计要点

1. **均衡而非散乱**：三模块均衡的前提是每一步都有清晰的短期目标——探索时有"下一处该解锁的地形/赐福地点"，战斗时有"这一场冲几星"，养成时有"这只该进化/造化"。
2. **刻意区别于参考作**：移动能力不照搬通用技能，而是用**中式神话灵兽能力**承载（详见探索系统）。
3. **战斗评分是循环引擎**：战斗不只看输赢，"怎么打赢"（回合数、伤亡、特殊击杀）决定养成资源品质。
