# 道具与装备设计

## 一、装备槽位总览

| 类别 | 槽位数 | 说明 |
| --- | --- | --- |
| 武器 | 1 | 只能装备一把 |
| 防具 | 5（头部 / 衣服 / 裤子 / 手 / 脚） | 各部位除基础防御外有额外特色 |
| 饰品 | 4（戒指 ×2 / 香囊 / 腰带） | 提供属性 / 特殊效果 / 相性加成 |
| 坐骑 / 代步 | 1 | 影响赶路等（具体待定） |

---

## 二、武器

- **只能装备一把**。
- **兵器类型决定可用武技**（见 `02_功法武学`：武技需装备对应兵器）。
- 提供攻击、出招速度、真气效率等基础属性。
- **战斗中从背包切换武器需要读条**（切武器有代价，非瞬时）。

---

## 三、防具（5 部位，各有特色）

| 部位 | 基础 | 特色示例 |
| --- | --- | --- |
| 头部 | 防御 | 可加**神（神思）上限** |
| 衣服 | 防御 | 特色待定（如精血 / 护盾） |
| 裤子 | 防御 | 特色待定 |
| 手 | 防御 | 可加**施展速度** |
| 脚 | 防御 | 可加**移动速度** |

---

## 四、饰品（4 槽）

- **戒指 ×2、香囊、腰带**。
- 提供属性、特殊效果、属性相性 / 功法增益等（偏特色加成）。

---

## 五、坐骑 / 代步（1 槽）

- 影响赶路（跨区域移动）等；具体效果待定。

---

## 六、品阶

- 装备品阶 **1–9**，与武学体系一致（**9 品最低，1 品最高**）。

---

## 七、强化与套装

- **强化**：可给武器 / 护具**添加词条**；**每件装备只能强化一次**。
- 部分词条可形成**套装效果**。
  - 示例 · 五毒套装：五个部位防具分别附加一种毒抗性 → 凑齐套装加成：**毒抗性加强**。

---

## 八、锻造手法（特殊词条）

- 锻造时可选择特殊**手法**，使武器带上**特殊词条（不可更改）**。
- 示例：锋锐、轻盈、厚重等。

---

## 九、其他道具（概述，后期细化）

- 消耗品（丹药 / 食物）、材料、书籍 / 秘籍、任务 / 特殊道具、背包 / 仓库。
- **神器碎片**：主角专属成长（见轮回与神器碎片成长线）。
