# 人物关系与社交设计

> 承接 `08_世界地图` 的双层好感之「**个人好感**」层（对具体 NPC）。

## 一、关系氛围（9 档）

- **4 正 + 陌生 + 4 负**，中间为陌生。
- 命名为**暂定**，最高档「至死不渝」按关系类型显示不同描述。

| 档位 | 暂定名称 | 说明 |
| --- | --- | --- |
| +4 | 至死不渝 | 最高档；按关系显示（义结金兰 / 夫妻 / 义子义父…） |
| +3 | 知交 / 莫逆 | **正面封顶**：不结特殊关系无法继续上升 |
| +2 | 投契 | |
| +1 | 友善 | |
| 0 | 陌生 | 中间 |
| -1 | 提防 | |
| -2 | 厌恶 | |
| -3 | 仇视 | |
| -4 | 血海深仇 | 最低档；可能主动寻仇 |

---

## 二、进阶门槛（特殊关系）

- 正面关系到 **3 档（知交）后封顶**，不再自然上涨。
- 进入 **4 档**需先在 3 档时**结成特殊关系**：**义结金兰 / 结为夫妻 / 收为义子 / 义父**等，之后才能继续提升至「至死不渝」。

---

## 三、互动方式（提升 / 降低）

> 均纳入；具体环境与数值**后期随互动环境确定**。

- **正向**：谈话 / 拜访、赠礼（画 / 礼品 / 稀物）、切磋 / 比武、雅事互动（对弈 / 抚琴等）、助人 / 事件拉选、结义 / 婚姻 / 收徒。
- **负向**：盗窃、伤害、夺宝、抉择得罪等。

---

## 四、特殊关系与衔接

- **主角**：可**义结金兰 / 娶嫁 / 收徒**（注意：**玩家不拜师**，但可收徒）。
- **NPC 之间**：也会**结亲生子**，联动**村庄人口增长**（见 `07_村庄经营`）。
- **婚姻 / 收徒收益**：同伴 / 助战 / 经营 / 加成等（**待定**）。

---

## 五、关系作用

- **正面**：交友获功法、援助 / 助战、赠礼回礼、结义 / 婚姻 / 收徒、相关事件、村庄招募。
- **负面**：敌对、被偷袭 / 寻仇、拒绝交易交流；血海深仇者可能主动来犯。

---

## 六、与其他系统衔接

- **世界 / 门派**：个人好感与门派恩义并行（双层）。
- **技艺**：赠礼（画 / 礼品）、雅事（琴 / 棋）提升好感。
- **村庄**：NPC 婚育补充人口 / 人力。
- **功法获取**：高好感 → 交友获功法。
- **事件**：社交事件、人物支线。
