# 核心玩法与游戏循环

## 一、一句话定位

武侠 / 仙侠题材的**角色养成 RPG**：玩家扮演一名"器灵转世者"，在被「阴煞」侵蚀的世界中**综合养成、探索净化、收集神器碎片**，逐步揭开身世真相，最终开启酆都、终结灾厄。

---

## 二、核心循环图

```
据点（家园）养成        ←——— 可选：经营村庄产出资源
      │
      ↓ 出发（跨区域移动 = 消耗时间）
开放世界探索（建驿站解锁区域；区域内自由移动，不耗时）
      │
      ↓
解决区域事件 / 战斗 / 奇遇 → 获取资源 · 功法 · 神器碎片 · 人脉
      │                         ↘ 降低该区域侵蚀度（有下限，随时间回升）
      ↓
回据点：综合养成（属性 / 功法 / 装备 / 技艺）
      │
      ↓ 变强
挑战更深区域 → 击败六大鬼王 → 开启酆都（主线终点）

【贯穿全程】收集神器碎片；死亡自动轮回、依碎片获加成，越轮回越强
```

---

## 三、时间模型

- **与现实时间无任何绑定**，玩家不行动时游戏时间静止。
- **时间仅在"做事"时流动**：读书、锻造、跨区域移动等会消耗时间。
- **同区域内移动不消耗时间**（区域内自由走动）。
- **过月结算**：跨月时统一计算 NPC 行动、事件生成、区域侵蚀度变化等。

---

## 四、开放世界与区域

- **驿站制**：除酆都外，玩家**花费资源建立驿站**后即可前往该区域，路线开放、非线性。
- **侵蚀度驱动**：越靠近酆都，区域侵蚀度越高；区域按当前侵蚀度生成**对应等级的事件与敌人**。
- **净化是动态的**：解决区域事件可降低侵蚀度，但**有下限**，且**一段时间后逐渐回升**至原侵蚀等级——保证区域可重复产出、世界不会被"刷空"。
- **主线门槛**：酆都临近区域分布**六大鬼王**，**集齐击败六大鬼王**方可开启酆都最终区域。
- **两类"变好"**：日常区域净化是**临时**的；主线里程碑（击败鬼王、最终救世）带来**永久**的世界改善。

---

## 五、养成（综合养成）

- **多线并重**：属性、功法、装备、技艺多条养成线，玩家**自由选择**培养方向。
- **产出途径**：读书、修炼、历练（战斗/探索）、事件奇遇各有产出，互为补充。

### 功法获取方式（无"拜师"机制）

- **门派恩义（主要途径）**：完成门派区域的事件 → 积累该门派**恩义** → 消耗恩义兑换功法（另有其他限制，详见门派 / 功法模块）。
- **正面手段**：与门派/人物交友等，换取功法。
- **负面手段**：盗窃等手段获取功法。
- 明确**不设拜师**这一获取路径。

---

## 六、成长特殊轴（贯穿全程）

- **神器碎片**：探索中收集，解锁 / 强化特殊能力。
- **轮回加成**：主角死亡即自动转世，依据已收集碎片获得加成（正面特质强化 / 负面特质清除 / 特殊物品等）。死亡是**容错与长线成长**，而非惩罚。

---

## 七、据点（村庄）：可选玩法

- **非核心、可完全不参与**：村庄经营是增益型可选系统，玩家不经营也能正常推进主线与养成。
- **家园定位**：村庄是剧情家园与安全区（主角继任村长，详见世界观）。
- **远程运营**：解锁**「远程通信（传书）」与「空间仓库」**能力后，玩家身处其他区域（如门派内）也能**远程下达村庄建设任务、调取村庄资源**，无需亲自返回。
- 定位：为愿意投入的玩家提供稳定资源与养成燃料，但不构成强制循环。

---

## 八、循环节奏小结

- **微循环**：进入区域 → 区域内自由活动解决事件/战斗 → 获取产出 → 降侵蚀。
- **中循环**：跨区域探索（耗时）→ 回据点养成 → 变强 → 解锁/深入更高侵蚀区域。
- **宏循环**：逐区净化与发展 → 击败六大鬼王 → 开启酆都 → 终结灾厄；全程收集碎片、轮回渐强。
