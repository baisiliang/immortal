## 补充概念

### 一、属性体系（六属性 + 阴阳 + 混元）

> **底层概念，不直接展示给玩家**：六属性（风、火、雷、水、木、土）是设计层的底层划分；面向玩家时**用更具武侠味的别名 / 描述**呈现（如「玄阴」「金刚」等门派化、功法化的称谓），而非直白的元素名。

- 阳（天）：风、火、雷
- 阴（地）：水、木、土
- 混元：所有属性均衡
- 无属性：锻体 / 体修（特殊流派）
- 基调：**武侠为主、仙侠元素为辅**，属性表现偏内力 / 真气 / 武学路数，而非直白的"元素魔法"。

**相生相克（待定，给出一个候选方案）**
- 同源（阳/阴）内部相生，阴阳之间相克。
- 候选关系（可调）：
  - 阳：风助火、火生雷、雷引风（阳内循环）
  - 阴：水生木、木固土、土纳水（阴内循环）
  - 阴克阳、阳克阴的具体对位（如水克火、土克雷、木克风…）由战斗组定稿。
- 混元：无明显克制，但上限更高、培养更难。
- 无属性体修：不吃属性相克，靠肉身硬抗与神魂。

### 二、阴煞（特殊侵蚀属性 / 世界灾厄之力）

> 独立于常规六属性，是世界观核心灾厄（详见 `世界观.md`）。

- **阴煞伤害**：敌方（被侵蚀者/群鬼）常用，可附带"侵蚀"减益。
- **侵蚀值（待定机制）**：玩家深入侵蚀区会累积阴煞侵蚀值；过高则触发减益甚至异化。
- **净化 / 克制**：阳属性（尤其雷法）、佛门、符箓等克阴煞；需净化道具/功法压制侵蚀值。
- **主角特殊**：器灵转世之身，对阴煞或有特殊亲和/抵抗（与神器碎片觉醒挂钩，待定）。

### 三、精气神（三元核心资源）

> 角色的三条核心资源，以武侠「精气神」统称。每项有 `当前值 / 上限`，上限随属性、境界、资质成长。

| 资源 | 对应 | 说明 |
| --- | --- | --- |
| **精（精力）** | 行动力 | 战斗外日常行动消耗：读书、研习、锻造等（**研习不消耗神**） |
| **气（气血）** | 血条（HP） | 生命，战斗承伤；归零则重伤 / 死亡 |
| **神（神魂）** | 蓝条（MP） | **纯战斗资源**：施展招式 / 功法的能量（战斗外不被研习消耗） |

- **不单设"内力"**：施展招式的能量即「神」（蓝条）。
- **恢复（建议）**：精力靠休整 / 睡眠（少量耗时）或丹药；气血靠休养 / 丹药 / 功法；神（蓝条）靠调息打坐 / 丹药 / 战斗中调息。

### 四、护盾与伤害结算

- **护盾**：由功法 / 装备提供的可恢复外层，**优先于气血承伤**。
- **气血**：护盾破后承伤的本源生命。
- **神魂攻击 / 直接伤害**：无视护盾与防御属性，直击气血（或扰乱神），是穿透性威胁；亦用于抵御"心魔 / 疯魔"类异化的抗性判定。

### 五、主角成长特殊轴（器灵转世）

- **神器碎片**：游玩中收集，解锁/强化特殊能力（可能对应额外属性或机制）。
- **转世觉醒**：随进程逐步觉醒器灵之力，作为区别于普通门派功法的独有成长线。
