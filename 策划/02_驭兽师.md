# 驭兽师（玩家角色）

驭兽师不是旁观的管理者，而是**队伍体系的一员**，自身有成长线，并直接卡着收集的上限。

## 一、修为境界

驭兽师走**修为/境界**成长路线（偏修仙基调），境界与灵兽品阶体系相呼应。

- 境界提升带来：
  - 驭兽空间**数量上限**提升（与装备叠加）。
  - **可驾驭品阶上限**提升（能驭使更高阶灵兽）。
- **待定**：境界序列命名与阶数；境界提升的方式（战斗经验 / 任务 / 修炼资源）。

## 二、加点与技能

每次境界提升（或对应节点）可**加点**，解锁**被动 / 主动技能**，可在多个方向上自定义，例如：

- **扩容类**：提高驭兽空间数量 / 提升可驾驭品阶。
- **辅助类**：战斗中治疗灵兽、提供增益。
- **强化类**：提高灵兽属性（如增攻、增防、增某属性）。
- **队伍体系类**：参与队伍搭建，提供与队伍构成相关的增益（具体体系待定）。

设计意图：让驭兽师成为 3V3 之外的"第四个变量"，加点流派影响整体队伍策略。

## 三、战斗中的驭兽师

- 作为**第四出战位**参战，不占 3 个灵兽出战位，有**独立行动条**。
- 可**使用道具与主动技能**，但**不能主动攻击**。
- 基础技能：鼓舞 / 治疗类（名称与功能待定）。详见 `03_战斗系统.md`。

- **待定**：
  - 加点是树状还是线性、是否可洗点。
  - 驭兽师基础技能与加点技能的具体数值、解锁节点。
