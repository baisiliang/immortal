# 世界观

## 一、主题

**轮回 & 长生**（详见草稿 `a.md`）。反派为求"长生"逆乱轮回，主角因"轮回"重生而崛起，二者形成主题对照。

---

## 二、核心背景故事（时间线）

**序章 · 地府大战**
- 某时代某年（**待定**），一位/一群神话传说中的强大存在（**待定**）为求「长生」入侵地府。
- 地府倾力反击，入侵者被镇压，长生计划失败。
- 但大战留下**隐患**：**轮回台**被阴煞侵蚀，埋下祸根。
- 大战中，地府一件重要法宝被打碎，**器灵流散转世为人**——即玩家主角。

**隐患爆发（大战 xx 年后，本作开始）**
- 被侵蚀的**轮回台发生异变**，地府大量"工作人员"（鬼差/判官等）被阴煞侵蚀。
- 地府爆发**暴乱**：**生死簿被毁、孟婆桥断、轮回停摆**。
- **群鬼出逃**，扰乱人间。

**阴煞侵蚀世界**
- 「**阴煞**」自酆都向外侵蚀世界，各地接连发生侵蚀事件。
- **人与动物均会被侵蚀**，被侵蚀者**性情大变、六亲不认、状若疯魔、嗜血嗜杀**。
- 越靠近酆都（中心），侵蚀越重、敌人越强（同心圆难度）。

**十八门派共谋救世**
- 天下**十八门派暂时放下恩怨**（仅表面），共同探求救世之路——暗流涌动。

**主角 · 器灵转世**
- 主角为大战中破碎法宝的**器灵转世之身**。
- 游玩过程中**收集散落的神器碎片**，并借助**转世**逐步觉醒/获取特殊能力。
- **死亡即轮回**：主角死后自动转世重生，依据已收集的神器碎片获得加成——如**正面特质强化、负面特质清除、获得特殊物品**等。区别于"换代继承"：传承的是同一个"器灵之魂"，越轮回越强。

## 三、开场叙事（序幕）

> 叙事基调：**不一上来就揭示苦大仇深的真相**，而是通过事件**一点点揭开主角身世与世界真相**（少年寻路、渐入真相的成长式叙事）。

1. **被收养**：主角自幼被**村长收养**，在边缘安全区的村庄平静长大。
   - 隐藏伏笔：村长实为**仙人化身**（仙人指路），村庄本身亦是仙人化身所设，暗中指点主角。**仙佛作为彩蛋**，其真实身份留待**后续支线**揭开。
2. **村庄遇袭**：长大后，村庄遭遇**阴煞侵蚀事件**。
3. **能力觉醒**：危难之际，发现主角**能够净化阴煞**，解救被侵蚀的村民。
4. **村长"陨落"**：危难中村长"去世"（**假死**），主角因特殊能力**继任新村长**。
5. **真相与使命**：村长临终（伪）告知主角世界被阴煞侵蚀的真相——**若不铲除源头，村庄终将覆灭**。为守护村庄，主角踏上**救世之路**。

> 动机设计：主角的出发点是**保护家园（村庄）**这一具体而温情的理由，而非天降大义，降低开场的沉重感。

---

## 四、叙事基调与难度取向（设计取舍）

- **整体基调偏轻松**，避免过度沉重。
- **不采用"主角越强 → 敌人越强 → 世界侵蚀度随主线反升"的反向难度设定**。
  - 那种设定会让玩家产生"我不去推进主线、世界反而更安全"的负反馈悖论。
  - 本作应让**玩家变强 = 世界向好**：主角推进主线、净化/铲除阴煞源头，**世界侵蚀度应随之下降**，给予正反馈与成就感。
- 难度仍由**空间**承载（同心圆：越靠近酆都越危险），而非随时间/主线进度惩罚玩家。

## 五、已定关键设定

| 项目 | 设定 |
| --- | --- |
| 灾厄之力命名 | **阴煞** |
| 灾厄起因 | 长生入侵 → 镇压留隐患 → 轮回台异变 → 地府暴乱 → 轮回停摆 → 阴煞侵世 |
| 被侵蚀表现 | 性情大变、六亲不认、状若疯魔、嗜血嗜杀（人、动物皆可被侵蚀） |
| 地图中心 | 酆都（地府入口），同心圆难度递增 |
| 玩家身份 | 地府破碎法宝的**器灵转世**，集神器碎片 + 转世觉醒；**死后自动轮回、依碎片获加成** |
| 开场动机 | 被村长（仙人化身）收养 → 村庄遇袭 → 觉醒净化之力 → 继任村长 → 为护村救世 |
| 叙事方式 | 渐进揭示身世/真相，开场轻松不沉重 |
| 难度取向 | 难度由空间承载（同心圆）；**主线推进使世界变好、侵蚀度下降**，不走"越强越难"的反向难度 |
| 门派 | **18 个**，需做**门派好感度系统** |
| 地理参考 | 重庆版图，古代神话/传说地点为主 |

2. **入侵者身份**：神话传说中求长生的强大个人或团体是谁？（候选：上古大巫、求长生的方士集团、逆乱天命的魔头、某位失败的"仙"等）

8. **门派好感度系统**：好感度如何积累/影响（习功法、援助、结局走向等），详见架构·人物关系系统。

## 八、同心圆侵蚀分区

> 从外到内：

1. **边缘安全区**（出生地·主角村庄）：未被侵蚀，正常城镇，新手区。
2. **侵蚀前沿区**：阴煞渗透，半沦陷村镇，中等难度，正派门派在此抵抗。
3. **重度侵蚀区**：地貌异化、生灵化魔，高难度，沦陷门派遗址在此。
4. **核心 · 酆都**：地府入口，最终区域，阴煞源头与轮回台所在，主线终点。
