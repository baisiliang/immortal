# 核心循环 · 决策记录

## 已定

- **时间模型**：与现实无绑定，玩家不行动则静止；仅"做事"（读书 / 锻造 / 跨区域移动等）流动时间；**同区域内移动不耗时**；**过月结算**（NPC 行动、事件生成、侵蚀度变化）。
- **养成引擎**：综合养成，属性 / 功法 / 装备 / 技艺多线并重，自由选向。
- **功法获取**：门派恩义兑换（主）、交友（正面）、盗窃（负面）；**不设拜师**。
- **开放世界**：除酆都外，花资源建**驿站**即可前往（开放、非线性）。
- **区域侵蚀度**：越近酆都越高；按侵蚀度生成对应等级事件 / 敌人；解决事件可降侵蚀度，但**有下限**且**随时间回升**。
- **主线门槛**：酆都临近区域有**六大鬼王**，击败六大鬼王方可开启酆都（终点）。
- **难度取向**：不走"越强越难、侵蚀随主线反升"；主线推进使世界**永久变好**、区域净化为**临时**；难度由空间（同心圆）承载。
- **村庄定位**：**可选**玩法（非核心，可完全不参与）；解锁"远程通信（传书）+ 空间仓库"后可远程下达村庄建设任务、调取资源。

## 备注

- 叙事采用渐进揭示（被村长 / 仙人化身收养 → 村庄遇袭 → 觉醒净化 → 继任村长 → 救世）。
