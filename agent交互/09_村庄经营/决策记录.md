# 村庄经营 · 决策记录

## 已定

- **定位**：可选玩法、**纯安全区（不被袭击）**、可远程运营（传书 + 空间仓库）、由各职业经营建筑组成。
- **规模等级**：村庄分等级，每级有**人力上限**；升级提上限 / 解锁建筑。
- **人力系统**：
  - 成年村民提供**人力**（抽象值，无需对应具体村民）。
  - 建筑经营需**分配人力**；**分配有序**，人力不足时后分配者被撤回、该建筑停产。
  - 招人建筑等级跟上村庄等级时，不会大面积缺人力。
- **人口增长**：特殊建筑**招募流民** + 村民**概率结婚生子**。
- **产出**：经营建筑**每月结算产出一次**，反哺养成 / 装备 / 商贸 / 净化。

## 待定

- 人力数值与产出公式。

- 远程运营解锁条件与限制。
