# 功法武学 · 决策记录

## 已定（武学五类，参考通行武侠分槽但命名自定）

| 类别 | 装配规则 | 作用 |
| --- | --- | --- |
| 心法 | 只能运行 1 部（核心） | 内功根基，被动属性、灵气（神 / 蓝）上限 |
| 护体 | 只能运行 1 部 | 运转自动耗灵气生成护盾 + 附带效果 |
| 身法 | 可随时切换 | 移动 / 走位加成 |
| 武技（暂定，原"催破"，待换名） | 随时使用，需装备对应兵器 | 直接造成伤害，无特效 |
| 绝技 | 可装备 | 装备后提供特殊效果 |

## 已定（等级与篇章）

- 武学品级 1—9：**9 品最低，1 品最高**。
- 一部武学横跨若干品级，按**篇章**划分，每篇章对应一个品级。
  - 例：基础心法范围 9—7，分 9 / 8 / 7 品三个篇章。
- 晋级：学完某品篇章 → 可用该品级武学，并解锁学习下一品篇章，逐篇精进。
