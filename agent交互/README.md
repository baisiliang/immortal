# Agent 交互归档

本目录记录与 AI 协作过程中的**关键决策与结论**，按主题分子文件夹归类。目的：

- 后续交互**只需读取相关主题的记录**，无需把全部历史传入上下文。
- 决策留档、可追溯、方便查阅。
