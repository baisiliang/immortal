# 世界地图与门派 · 决策记录

## 已定

- **地图**：以重庆版图为参考，神话传说地点建门派；古代题材为主，少量外来门派（如苗族巫蛊）占比小。
- **侵蚀度同心圆**：越近酆都越高，区域事件 / 敌人按侵蚀度分级；解决事件降侵蚀度（有下限、会回升）。
- **移动 / 驿站**：跨区域耗时（神思）；建驿站可达；酆都需击败六鬼王开启。
- **门派**：共 18 个，立场（正 / 邪 / 亦正亦邪）+ 地域神话特色；**一世只能加入一个门派**。
- **关系双层**：①地区 / 门派**恩义**（贡献值，完成门派区域事件积累，换功法 / 资源）②个人**好感**（对 NPC，影响交友 / 送礼 / 事件）。
- **功法获取**：恩义兑换（无残篇）/ 交友 / 盗窃 / 加入门派传承（传道师消耗贡献授传承，依等级 + 资质，可能残篇）；书可修补残篇。

## 18 门派名单（草案，原创古风名，流派通俗，避免撞经典）

正派 / 佛道：霁月庵、云岚观（道）、紫府雷宗（道·缙云山）、莲台寺（佛·金佛山）、瑶光峰（女）、夔龙剑派、百衲帮（江湖）。
中立 / 隐逸：玄机阁（暗器毒）、绛雪谷（毒医）、采薇山庄（隐逸医农）、朝云阁（巫山·女）、杏林坞（医）。
亦正亦邪 / 外来：裂石堂（刀）、沧浪寨（水）、傩神坛（土家·驱邪）。
邪派：万蛊教（苗疆蛊）、赤煞门（刀·邪）、幽煞门（借阴煞）。
> 丰都 / 酆都为阴煞中心、敌方，不作玩家门派。
