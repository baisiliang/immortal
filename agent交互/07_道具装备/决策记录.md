# 道具与装备 · 决策记录

## 已定

- **槽位**：武器 ×1、防具 5 部位（头 / 衣 / 裤 / 手 / 脚）、饰品 ×4（戒指 ×2 / 香囊 / 腰带）、坐骑 / 代步 ×1。
- **武器**：只能装备一把；兵器类型决定可用武技；**战斗中切换武器需读条**。
- **防具特色**：头=神（神思）上限，手=施展速度，脚=移动速度；衣 / 裤特色待定。
- **饰品**：偏特殊效果 / 属性相性 / 功法增益。
- **品阶**：1–9（9 低 1 高，与武学一致）。
- **强化**：给武器 / 护具加词条，**每件只能强化一次**；部分词条凑成**套装效果**（例：五毒套装→毒抗加强）。
- **锻造手法**：锻造时选特殊手法 → 武器带**特殊词条（不可更改）**，如锋锐 / 轻盈 / 厚重。

## 设计取向

- 不走"随机数值堆砌"路线；装备偏**定位清晰 + 服务武学流派 / 相性 + 套装特色**。

## 待定

- 衣 / 裤特色、切武器读条时长、强化词条池与获取、套装清单、坐骑效果、锻造手法清单。
