# 决策记录 · 核心循环与系统框架

对应策划文档：`策划/00_核心循环.md`、`01_灵兽系统.md`、`02_驭兽师.md`、`03_战斗系统.md`、`04_探索系统.md`

## 已定

- **主轴**：三大模块（探索 / 战斗 / 收集养成）**均衡环形互喂**，不设单一主轴。
- **探索解锁**：混合门槛 —— 中式神话灵兽能力（如猴王火眼金睛看破虚妄）为主路，辅以 Boss 实力门槛、解谜道具门槛。能力刻意区别于参考作的通用化技能。
- **战斗评分**：每场战斗都评分，维度为**回合数 + 伤亡**，星级决定掉落概率/品质。
- **取消"连携 Combo"** 概念（已从整体思路移除）。
- **特殊击杀掉落**：用特定方式击败特定灵兽，概率掉特殊道具（参考仙剑系列尸块设定）。
- **驭兽空间双重限制**：数量上限（驭兽师境界 + 装备）+ 可驾驭品阶上限（驭兽师境界）；特殊道具可越级驾驭。
- **驭兽师成长**：走修为/境界路线，并可加点解锁被动/主动技能（扩空间、战中治疗、增灵兽属性、参与队伍体系），是队伍的一员。
- **造化（赐福）**：参考转职思路，灵兽到一定等级在特定洞天福地接受某"类"造化（例：峨眉山水帘洞→灵长类）。与进化（血脉蜕变）是两条不同成长线。中式叫法**暂定"造化"**（备选：敕封/受箓/钟灵）。
- **世界地图**（见 `策划/05_世界地图.md`）：
  - 一张**连通大图（纯银河城）**。
  - 分区**取材中式神话地点、另起别名**，每区灵兽与当地传说对应（神农架有龙、峨眉山有猴）。
  - **多个城镇**承担管理功能，无单一大本营。
  - **开局可选出生城镇**，决定初始区域与初始灵兽，带重玩价值。
- **战斗骨架**（见 `策划/03_战斗系统.md`）：
  - 站位**参考《暗黑地牢》**：技能有位置限制，大部分技能优先打前排，位移成为策略。
  - 行动**带速度的回合制（参考仙剑三）**：速度快者一轮可多次行动；部分法术需读条、可被打断。
  - 驭兽师为**第四出战位**：有行动条、可用道具与主动技能、**不能主动攻击**，有鼓舞/治疗类基础技能（名称功能待定）。

## 待定

- 世界地图：区域清单/别名/对应传说与灵兽谱系；城镇功能分布与传送；出生点清单与初始灵兽；**多出生点 × 连通大图的防卡死方案（软门槛 vs 多入口平衡）**。
- 九阶灵兽称号完整序列与排序。
- 驭兽师境界序列命名与提升方式。
- 造化：具体属性加成、成长曲线、是否可叠加/切换/洗点、类别与洞天福地对应清单、等级门槛。
- 进化：所需道具、血脉条件、可进化谱系。
- 战斗：行动顺序、站位规则、驭兽师在场参与形式；评分权重与星级阈值；掉落表。
- 特殊击杀：触发条件、对应灵兽与掉落物、概率。
- 驭兽空间：是否区分出战位/仓库位；装备加成数值。
- 中式叫法定稿："造化"是否采用。

## 讨论备注

- 备选主轴方案（探索为主轴 / 收集为主轴 / 战斗为主轴）均被否决，选择均衡环形。
- 移动能力解锁明确不照搬怪物圣所的通用技能，改用中式神话灵兽能力承载。
