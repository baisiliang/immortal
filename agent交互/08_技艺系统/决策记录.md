# 技艺与生活技能 · 决策记录

## 已定（最新框架）

- **两大类**：制造、研学。
- **每个方向自成一职，共 12 个**（制造 6 + 研学 6）；职业名**后定**（暂用方向名）。
- 其中「**制茶 + 酿酒**」合为同一个方向 / 职业。
- 每个方向（职业）= **技能 + 被动 + 经营建筑**。
- **经营建筑**：每月结算**产出一次**；各建筑**特色后定**。研学类建筑侧重**个人被动增益 / 训练加速**；炼丹（丹房）为产出型。
- 技艺资质决定学习速度与上限；方向随等级解锁 / 强化（进阶可选学）。

### 制造类方向（6；被动统一：与制品效果 / 词条 / 等级相关）
- 锻造（兵防 + 锻造手法词条）、制饰（腰带 / 香囊 / 戒指）、制药、制毒、做菜（非战时增减益）、制茶酿酒（茶 + 酒，短期加成 + 后负面，可作礼品）。

### 研学类方向
- 琴（抚琴·低好感提好感｜神恢复 / 上限）
- 棋（对弈·小游戏可下注｜特殊卡牌 / 技能）
- 书（修补残页 = 修补功法残篇｜研学 / 阅历加成）
- 画（作画送礼提好感｜魅力 / 送礼好感）
- 炼丹·道法（恢复 / 延寿｜延寿｜产出型）
- ○佛法（诵经 / 禅修·净化 / 降心魔 / 安神｜抗心魔 / 降疯魔）
